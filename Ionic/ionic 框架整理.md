### 目录结构
```
css                     // 存放css文件  
images                  // 存放图片文件  
js                      // 存放js文件  
    - controllers       // 存放控制器文件  
    - directivers       // 存放指令文件  
    - lib               // 存放外部包：ionic、require等  
    - services          // 存放服务文件  
pages                   // 存放页面文件  
app.js                  // angularJs启动文件  
bower.json              // Js管理  
index.html              // html入口  
router.js               // 记录路由相关文件  
```
